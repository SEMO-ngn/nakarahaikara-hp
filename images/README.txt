【画像の差し替え方法】

このフォルダに、以下のファイル名で写真を入れるだけで
サイトの背景に自動的に反映されます（コードの変更は不要です）。

hero.jpg        … ファーストビュー背景（店内 or 料理の看板写真）横長推奨
menu-beef.jpg   … 牛ホホ網焼き
menu-pork.jpg   … 信州ポーク鉄板焼き山盛り
sports.jpg      … スポーツ観戦・プロジェクターの様子
insta-1.jpg 〜 insta-6.jpg … Instagramセクションのタイル（正方形推奨）
ogp.jpg         … SNSシェア時に表示されるサムネイル画像（1200x630推奨）

推奨サイズ：
・hero.jpg / sports.jpg      … 1600px以上（横長）
・menu-beef.jpg / menu-pork.jpg … 1400px以上（横長）
・insta-1〜6.jpg             … 800px以上（正方形）

画像が未設置の間も、雰囲気に合わせたグラデーション背景が
自動表示されるため、レイアウトが崩れることはありません。
